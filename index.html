Feature Callout

The Radii Image Library (RIL) is a C++ library that abstracts image representations from algorithms and allows writing code that can work on a variety of images with performance similar to hand-writing for a specific image type.

RIL makes it Easy!

With RIL abstrations, image-frame processing is fairly intuitive and easy.

Using Radii Imaging Library (RIL) Commercially

To make way for Open Research, we at Radii School of Thoughts have collaborative sessions and internships which helps us with maintaining RIL free for scientific research. Any alterations to the library must forgo Product Licensing.
